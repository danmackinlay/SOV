# Session log

A log of every audition session — what we ran, what it cost, what we learned. The point is institutional memory: a teammate joining in three months should be able to read this and understand what's been tried.

Each entry is short. If it warrants more space, write a separate doc and link it.

## Format

```
### YYYY-MM-DD — short description

- **Phase:** N
- **Provider:** RunPod / vast.ai / Lambda
- **Hardware:** e.g., 4× H100 80GB
- **Duration:** e.g., 3h 20m
- **Cost:** ~$X AUD
- **Run by:** name
- **What we did:** 1–3 lines
- **What we learned:** 1–3 lines, link to followups
```
